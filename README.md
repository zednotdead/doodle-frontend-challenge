# Doodle Frontend Challenge - Zuzanna Żołnierowicz

## Setup

In order to get this frontend challenge working, you need [Mise](https://mise.jdx.dev/).

Run these commands:

```bash
mise install
bun install
```

Next, copy `.env.example` to `.env`, and adjust the values inside to match your instance of [frontend-challenge-chat-api](https://github.com/DoodleScheduling/frontend-challenge-chat-api).

## Development server

To start the development server, run:

```bash
bun dev
```

## Building the application

In order to build the application, simply run:

```bash
bun run build
```

The application will be outputted to `./dist`.
